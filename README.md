# Mehndi Website

Welcome to the Mehndi Website project! This repository contains a collection of mehndi designs and the corresponding web application to showcase them.

## Features
- **Design Gallery**: A beautiful collection of mehndi designs.
- **User Submission**: Allows users to submit their own mehndi designs.
- **Responsive Design**: The website is fully responsive and accessible from mobile and desktop devices.

## Getting Started
To get a copy of this project up and running on your local machine, follow these steps:

### Prerequisites
- Web browser (Chrome, Firefox, Safari, etc.)
- Node.js (for running the development server)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Biswapriti/mehndi_website.git
   ```
2. Navigate into the project directory:
   ```bash
   cd mehndi_website
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```

## Usage
Open your web browser and navigate to `http://localhost:3000` to view the application. Explore the different mehndi designs!
